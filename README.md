# UR7e ArUco LLM

Voice-guided UR7e pick-and-place demo using ArUco markers, OpenCV, RTDE,
Robotiq Hand-E, Whisper, and an optional Ollama/LangChain selector.

The video demonstration: _demo_ur7e-aruco-llm.mp4_ and the presentation _presentation_ur7e-aruco-llm.pdf_

## Setup

Use Python 3.11 on Windows:

```powershell
python -m venv .venv
.\.venv\Scripts\activate
python -m pip install -r requirements.txt
```

## Network

Set a static IPv4 address on the robot in the UR Teach Pendant. This project
uses `10.20.3.136` in the runtime scripts.

Set the PC Ethernet adapter to the same subnet, for example:

```powershell
netsh interface ipv4 set address name="Ethernet" static 10.20.3.20 255.255.255.0
ping 10.20.3.136
Test-NetConnection 10.20.3.136 -Port 30004
```

Update `ROBOT_IP` constants in the scripts if your robot uses another address.

## Calibration And First Run

Run the helper scripts in order before starting the main app:

```text
000_find_opencv_camera_id.py
001_generate_charuco_board.py
002_calibrate_camera_charuco.py
003_test_ur_rtde_connection.py
004_generate_aruco_marker_sheet.py
005_collect_handeye_dataset.py
006_solve_handeye_calibration.py
007_teach_robot_positions.py
008_test_lift_markers_0_to_5.py
```

These steps create or verify `camera_matrix.npy`, `dist_coeffs.npy`,
`handeye_dataset.npz`, `handeye_result.npz`, and `robot_positions.json`.

After that, run:

```powershell
python main.py
```

The web panel opens at `http://localhost:8766/`. Ollama is optional; without it,
the voice selector falls back to local lexical matching.

`command.wav` is a temporary debug recording generated by `voice_select.py`
during voice selection.
